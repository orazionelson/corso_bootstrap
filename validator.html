Bootstrap validator

Validare i form lato client

http://1000hz.github.io/bootstrap-validator/

Bootstrap validator è un'estensione che ci permette di verficare e validare i campi di un form prima dell'invio.

Per installarla possiamo:

  • A) Scaricare lo script da GitHub e mettere il file validator.js nella nostra cartella dei javascript e quindi creare il link nel codice HTML della pagina.
  • B) Linkare lo script via CDN.

Analogamente a i javascritp nativi di Bootstrap anche Validator si attiva aggiungendo l'attributo data-toggle="validator" al form da validare.


  ...

Lo script tiene conto, per quanto possibile, della nomenclatura standard di HTML5 per i tipi di input:

  • type="email"
  • type="url"
  • type="number", di concerto con gli attributi min e max
  • pattern="Reg(ular )?Exp(ression)?"
  • required

Ci sono, inoltre, tre attributi non-standard evocati tramite data-*:

  • data-match="#inputToMatch", per gestire il match tra diversi input, ad esempio la conferma di una password
  • data-minlength="5", per forzare un campo ad essere compilato con un numero minimo di caratteri
  • data-remote="/path/to/remote/validator", per validare un campo a fronte di un file remoto con una chiamata via AJAX
