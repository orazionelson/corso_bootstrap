Se le icone incorporate in bootstrap non dovessero essere sufficienti esiste un ulteriore set open source: Font Awesome.

Il modo più semplice di incorporare font awesome è aggiungere la stringa del CDN appena dopo il link al CSS di bootstrap nella HEAD del documento HTML.

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css">

Per usare le icone di Font Awesome, basta aggiungere le classi di FA a un elemento vuoto
