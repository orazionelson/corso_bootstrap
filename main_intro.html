Tip: il server locale

Anche se il browser interpreta senza problemi HTML è sempre meglio richiamare il nostro progetto tramite un meccanismo client-server usando il protocollo http e non direttamente dal filesystem.

Per emulare il meccanismo client-server sul nostro pc locale possiamo usare:


Xampp / Lampp

  • Installare l’ambiente Xampp o Lampp.
  • Copiare il progetto nella directory radice/root del web-server locale e quindi puntare alla pagina: http://localhost/mioprogetto

Server-web minimale di Node.js

  • Installare Node.js e npm, cfr. https://docs.npmjs.com/getting-started/installing-node.
  • Installare un http server:
    • Con Windows, da amministratore, aprire cmd.exe o la Poweshell e dare il comando:
      npm install -g http-server
    • Con linux e mac, aprire un terminale e dare il comando:
      sudo npm install -g http-server
  • Quindi nella cartella del nostro progetto, sempre da terminale avviare il server: 
    http-server -c-1
  • A questo punto il nostro progetto con Bootstrap sarà raggiungibile all'indirizzo: http://localhost:8080 o http://127.0.0.1:8080
