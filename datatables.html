DataTables

DataTables è uno script che migliora la visualizzazione delle tabelle. Offre di default features come la paginazione, alcuni filtri, il passaggio dei dati via ajax e un'estensione per Bootstrap.

Per installare datatables è necessario incorporare nella pagina:

  • Il CSS: dataTables.bootstrap.css, dopo il CSS principale di Bootstrap
  • I Javascript: jquery.dataTables.js e dataTables.bootstrap.js dopo il JS di Bootstrap

//nella head
<link rel="stylesheet" href="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css">
<link rel="stylesheet" href="https://cdn.datatables.net/1.10.11/css/dataTables.bootstrap.min.css">
...
//dopo il footer
<script src="http://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/js/bootstrap.min.js"></script>
<script src="https://cdn.datatables.net/1.10.11/js/jquery.dataTables.min.js"></script>
<script src="https://cdn.datatables.net/1.10.11/js/dataTables.bootstrap.min.js"></script>

Assegnare un ID alla tabella da trasformare.

<table class="table table-striped" id="example" cellspacing="0" width="100%">

Chiamare dataTables via javascript:

$('#example').DataTable();
