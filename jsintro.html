Bootstrap.js

È possibile richiamare le quasi tutte le componenti javascript di Bootstrap usando gli attributi data-* nel mark-up. Senza una sola riga di javascript.

Ad esempio per collassare una elemento con id id="collapseExample" basterà governarlo con un bottone che avrà come trigger per il javascript data-toggle="collapse" e come obiettivo l'elemento stesso data-target="#collapseExample".
