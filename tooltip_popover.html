Tooltip & Popover

Mostra informazioni in un fumetto

Tooltip

Mostra al passaggio del mouse il valore dell'attributo title in un fumetto posizionabile rispetto all'elemento.

Come si attiva:

Nell'elemento:

  • Trigger: data-toggle="tooltip"
  • Posizionamento: data-placement="top|right|bottom|left"
  • Contenuto: title="Testo mostrato nel tooltip"

Attenzione: il javascript tooltip non si attiva tramite il solo attributo data-toggle, occorre una chiamata javascript del come riportato nell'esempio.

HTML

Javascript


$(function () {
	$('[data-toggle="tooltip"]').tooltip()
})
Tip: E' possibile personalizzare lo stile del popover con le classi .tooltip, .tooltip-title, .tooltip-arrow

Popover

Simile al tooltip ma le informazioni il fumetto con le info si attiva al click.

Come si attiva:

Nell'elemento:

  • Trigger: data-toggle="popover"
  • Posizionamento: data-placement="top|right|bottom|left"
  • Contenuto:
    • Titolo del fumetto: title="Testo mostrato titolo del popover"
    • Corpo del fumetto: data-content="Testo mostrato nel popover"

Attenzione: anche Popover va attivato tramite un'esplicita chiamata javascript del come riportato nell'esempio.

PopOver

HTML

Javascript


$(function () {
	$('[data-toggle="popover"]').popover()
})
Tip: E' possibile personalizzare lo stile del popover con le classi .popover, .popover-title, .popover-content
